# 🧭 PokeExplorer

A responsive Pokémon explorer web app built using React.  
Users can search, filter, view details, and save their favorite Pokémon.

🔗 Live Demo: https://pokeexplor.netlify.app/

---

## 🚀 Features

- 🔍 Search Pokémon by name
- 🧪 Filter Pokémon by type (Fire, Water, Grass, etc.)
- 📄 Detailed Pokémon page (image, stats, abilities, etc.)
- ❤️ Add / Remove Favorites (with localStorage)
- 📄 Pagination support
- ⚡ Fast and responsive UI

---

## 🛠️ Tech Stack

- **Frontend:** React (Hooks)
- **Routing:** React Router
- **Styling:** CSS
- **API:** PokéAPI
- **Build Tool:** Parcel
- **Deployment:** Netlify

---

## 📦 API Used

- https://pokeapi.co/api/v2/pokemon

Data fetched includes:
- Name
- Image (sprites)
- Types
- Abilities
- Stats

---

## 🧠 Key Concepts Implemented

- useEffect & async data fetching
- Promise.all for parallel API calls
- Controlled inputs (search & filter)
- Conditional rendering
- State management with useState
- LocalStorage for persistence (Favorites)
- Dynamic routing with React Router
